hyCalendar とは:

"ハイパーリンクを使えるカレンダー" です．
yyyy/m/d あるいは m/d 形式で記述された日付を自動的にリンクし，
ダブルクリックでその日付に移動することができます．
また，オプションとして，その日付にカーソルを合わせるだけで
その日付の予定をポップアップさせることもできます．

その一方で，予定として記述された文字列中の
URL (http(s), ftp, mailto など) に対してもハイパーリンクを設定します．

詳細な利用方法については，hyCalendar.chm を参照してください．

インストール方法:

アーカイブに含まれている以下のファイルを，任意のディレクトリに配置してください．

hyCalendar.exe  実行ファイル
hyCalendar.chm  ヘルプファイル
holidays.txt    日付データ


ファイル拡張子(.cal)の関連付けおよびその解除は，
設定ダイアログから行うことができます．
それ以外ではレジストリを使用しないため，アンインストール時は
ファイル関連付けを解除後，これらのファイルを削除してください．


このツールでできること:

- 予定の記入
   * 日付のセルをダブルクリック(あるいは Enter キー)で予定の記入．
   * 入力が終わったら別の場所をクリック(または，Alt+Enter あるいは ESC キー)． 
   * 記入された内容に含まれる URL は，セル選択時に
     ハイパーリンクが表示されます．また，右クリックしたときの
     コンテキストメニューからもハイパーリンクを利用できます．
   * ハイパーリンクはダブルクリックで移動可能です．

- 期間予定の記入
   * 日付のセルを右クリックして「新しい期間予定を追加」を選択．
   * ダイアログで，開始日付，終了日付，キャプション文字列，
     線の色，スタイル，などを指定することができます．
     文字列の色は，特に指定しない限り，フォントで設定された項目が有効です．

- 周期予定の記入
   * 周期予定とは，条件（例：毎月第一火曜日）に基づいて生成される予定です．
   * [カレンダー]-[周期予定の編集] を選択することで，
     周期予定を管理するためのダイアログが開きます．
     予定の追加，予定に対する条件の追加，編集ができます．
   * 条件としては，特定の日，週，曜日による指定と，
     祝日によるズレなどが設定できます．

- 日付のカウントダウン
   * 指定した日あるいは特定の周期予定までの日数を
     ステータスバーに表示することができます．
   * [カレンダー]-[日付カウントダウンの設定] から設定を行うことができます．

- 画面下に付属しているフリーメモ欄
   * 予定と一緒に保存されます．
   * フリーメモ欄の URL は，直接 URL をダブルクリックすることで
     開くことができます．

- TODO管理
   * 予定と一緒に保存されます．
   * フリーメモ欄右側に表示されています．
   * [カレンダー]-[TODO管理] あるいは，右クリックによる
     コンテキストメニューから編集できます．
   * TODO管理ダイアログから「チェック済みTODOを隠す」指定を行うことで
     終了したTODOをリストから隠すことができます．
   * この欄に記述されたURLも，ハイパーリンクの対象となります．
     シングルクリックでは，アイテム編集になるので注意してください．

- 予定の検索
   * メニューから [カレンダー]-[予定の検索] を選ぶことで，
     検索ダイアログが開きます．
     検索ダイアログでは，上端のコンボボックスに値を入力して Enter キーあるいは
     右側のボタンを押すと，検索結果が一覧表示されます．
   * 検索ツールバーのコンボボックスに値を入力して Enter キーあるいは
     右側のボタンを押すと，その文字列が予定メモ中に含まれている日の
     背景色が変わります．（もう一度ボタンを押すと解除されます）

- 設定の変更
   * すべての設定は，メニュー [オプション]-[設定] を選ぶことで開く
     設定ダイアログから行うことができます．
   * 設定は，EXE ファイルと同じディレクトリにある hycalendar.ini に
     保存されます．

- 設定のカスタマイズ
   * 表示フォント
   * ファイルの自動保存
      + プログラム終了時に自動的にファイルに保存するようになります．
        ファイルを保存したくない場合は [ファイル]-[ファイルを保存せずに終了] を
        選択してください．
   * ウィンドウ位置の保存
      + プログラム終了時のウィンドウ位置を ini ファイルに保存します．
   * ツールバー順序の保存
      + プログラム終了時のツールバーの位置・サイズを保存します．
   * .cal 拡張子の関連付け
     .cal 拡張子をこのアプリケーションに関連付けることができます．
     また，解除も設定ダイアログから実行することができます．
   * 表示サイズをページごとに区別する
     「表示サイズ」ボックスの設定値を，１ヶ月ごとに区別するようになります．

- 表示縮尺の設定
   * 「表示サイズ」ボックスに適切な値を "%" を付加して直接入力するか，
     リストから選択すると，セルの表示サイズが変化します．
     100% が指定されたとき，１ヶ月のデータがウィンドウに
     収まるように表示されます．

- 日付移動
   * ツールバー左端の日付入力欄に値を入力して，
     Enter キーを押下するまたは移動ボタンをクリックすることで，
     目標の日に直接移動できます．
     その日が含んだタブが開かれていない場合，自動で開きます． 
   * 不要なタブは，右クリックしたコンテキストメニューから
      [このタブを閉じる] を選択することで閉じることができます． 

- 切り取り，コピー，貼り付け，「元に戻す」
   * 選択された予定メモは，切り取り，コピー，貼り付けを行うことができます．
     予定メモに記入されている内容をテキストとみなして扱うため，
     フリーメモや他のテキストファイルなどと自由に移動することができます．

- 印刷
   * [ファイル]-[印刷] で印刷ダイアログが表示されます．
     指定した月の範囲で，１ヶ月１ページ単位のカレンダーを印刷します．

- その他
   * 予定のテキスト形式，CSV形式でのエクスポートが可能です．
   * 複数ファイルを重ねて表示することで，共有の予定ファイルなどを使用できます．
