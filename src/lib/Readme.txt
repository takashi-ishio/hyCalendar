【名    称】  印刷プレビュー制御コンポーネント plPrev Ver. 4.3 
【制作者名】  Mr.XRAY
=====================================================================

  印刷の描画コードを記述するCanvasと，そのCanvasに描画した内容をプ
  レビュー表示するためのカスタマイズ可能なプレビューフォームを提供
  するものです． Ver4.0では使用方法を簡単にしました．
  
  なお，Ver3.0から中村　拓男さんのDHGL 1.2を一部使用しています．
  
  マニュアルは同梱していません．インストール方法や使用方法について
  は以下のオンラインマニュアルを参考にしてださい．

   http://homepage2.nifty.com/Mr_XRAY/Delphi/plPrev/
   HGH03072@nifty.ne.jp
 
 
 [バージョンアップの注意]
 
 Ver4.0でOnCloseイベントを廃止しました．これは，プレビューフォーム関
 係のイベントをコンポーネントのイベントとして利用するようにしたためで
 す．代わりにOnFomrCloseを使用して下さい．
 (プレビューフォーム関係のイベント名の頭にはFormを付けています)
 
 後発の印刷関係コンポーネント
 　　　Query印刷コンポーネント Ver4.0以上
       DBGrid印刷コンポーネント Ver4.0以上
       タックシール印刷コンポーネント Ver4.0以上
 を利用する場合は，Ver4.1以上をインストールしてご利用下さい．
