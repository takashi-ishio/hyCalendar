■"HTML 404"とは

  HTMLファイルからリンクを抽出し、リンク先のファイルが存在するか確認します。
  インターネット上のファイルにも対応しています。 




■  動作環境

  Windows 98/2000/ME/XP で動作するはずです。 




■  インストール

  アーカイブを適当なディレクトリに解凍して実行してください。
  アンインストールはフォルダごと削除してください。
  レジストリには書き込んでいません。 




■  アンインストール

  ディレクトリごと削除してください。レジストリには書き込んでいません。
