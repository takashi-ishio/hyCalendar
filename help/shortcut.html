ショートカット一覧

ショートカット動作
Home, または CTRL + ←その週の先頭（左端）へ移動
End, または CTRL + →その週の末尾（右端）へ移動
Page Up一ヶ月前へ移動
Page Down一ヶ月後へ移動
CTRL + Page Up前のタブへ移動
CTRL + Page Down次のタブへ移動
CTRL + S[ファイル]-[上書き保存]
CTRL + P[ファイル]-[印刷]
CTRL + Q[ファイル]-[終了]
CTRL + Z[編集]-[元に戻す]
SHIFT + CTRL + Z[編集]-[やり直し]
CTRL + X[編集]-[切り取り]
CTRL + C[編集]-[コピー]
CTRL + V[編集]-[貼り付け]
SHIFT + CTRL + C[編集]-[ポップアップ表示される予定文字列のコピー]
CTRL + T[カレンダー]-[今日の予定へ移動]
CTRL + F[カレンダー]-[予定の検索]
CTRL + R[カレンダー]-[周期予定の編集]
CTRL + E[カレンダー]-[TODO管理]
F1[ヘルプ]-[hyCalendarのヘルプ]

[予定の検索]，[周期予定の編集]，[TODO管理]については， 設定ダイアログの[キー操作]タブで ショートカットキーの設定を変更することによって，ダイアログを開くだけでなく， 既に開いているダイアログを閉じることもできます．

以下は，TODOリストおよびTODO管理ダイアログ上でのみ有効なショートカットです．

ショートカット動作
CTRL + A新しい TODO の追加
CTRL + D選択した TODO の削除
CTRL + J選択した TODO を下に移動
CTRL + K選択した TODO を上に移動
F2選択したTODO の編集
