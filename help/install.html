hyCalendar のセットアップ

動作環境

Windows 2000, XP を対象として開発しています．

Windows 98 への対応は特に行っていません． バージョンによっては動くようですが，動作を保証してはいません．

インストール方法

アーカイブに含まれている以下のファイルを，任意のディレクトリに配置してください． 

hyCalendar.exe  実行ファイル
holidays.txt    日付データ
hyCalendar.chm  このファイル

hyCaledndar.exe をダブルクリックすると起動します． スタートメニューなどに登録する場合は，適宜ショートカットを作成してください． ショートカットを作成する場合は， カレントディレクトリが実行ファイルの存在するディレクトリとなるように設定してください．

拡張子の関連付け

hyCalendarは，".cal" をファイル拡張子として利用することができます． 拡張子の関連付けを行うと，作成したファイルを エクスプローラからダブルクリックすることで簡単に hyCalendar を起動できます．

拡張子の関連付けを行うには，hyCalendar を起動して，次の操作を行ってください．

  • メニュー [オプション]-[設定] を選択し，設定ダイアログを開きます．
  • [システム設定]タブにある [.cal拡張子をこのプログラムに関連付ける] ボタンを押します．

Windows Vista では，上記の操作は 管理者権限で実行する必要があるようです(Vista RC1 にて確認)．

アンインストール

hyCalendar が不要になってアンインストールしたい場合は， インストール時にコピーしたファイル群を削除してください．

拡張子の関連付けを行っていた場合は， 次の手順で関連付けを解除してください．

  • メニュー [オプション]-[設定]を選択し，設定ダイアログを開きます．
  • [システム設定]タブにある [関連付けを解除する]ボタンを押します．

誤ってファイルを先に削除してしまった（関連付けを解除していない）場合， hyCalendar の実行ファイルをもう一度配置し，上記の操作を実行後， 改めてファイルを削除してください．

設定のバックアップ・他のPCへの移行

hyCalendar の設定は，初回起動時に作成される hyCalendar.ini に保存されます． このファイルをコピーするだけで，設定を保存・移行することができます． 設定を初期化したい場合は，このファイルを削除してください．
